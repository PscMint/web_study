1. 对于一个div要出现滚动条，是父元素的div无法完全显示子元素的内容，因此通过设置父元素的overflow属性可以出现滚动条
   
2. 样式设定
```css
.scroll{
    height:200px;
    width:20%;
    border:1px solid;
    overflow: scroll;/*横纵滚动条*/
    overflow-x:scroll;/*横向滚动条*/
    overflow-y:scroll;/*纵向滚动条*/
}
.scroll::-webkit-scrollbar {/*滚动条整体样式*/
    /*高宽分别对应横竖滚动条的尺寸*/
    /*竖直滚动条宽度*/
    width: 30px!important;
    /*竖直滚动条宽度*/     
    height: 30px!important;
}
.scroll::-webkit-scrollbar-thumb {/*滚动条里面小方块*/
    
    border-radius: 0px;
    /*可以给滚动条设置阴影*/
    box-shadow:inset  0 0 5px rgba(231,20,20,0.3);
    background: rgba(231, 20, 20, 0.3);
    
}

.scroll::-webkit-scrollbar-thumb:hover {/*滚动条里面小方块可以加悬停效果*/
   background-color: rgba(231, 20, 20,0.5);
}
.scroll::-webkit-scrollbar-track {/*滚动条里面轨道*/
    box-shadow: inset 0 0 5px rgba(231, 20, 20,0.1);
    border-radius: 0;
    background: rgba(231, 20, 20,0.1);
}
```
